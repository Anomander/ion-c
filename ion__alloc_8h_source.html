00025 //
00026 // ion managed resources are broken into two general classes, primary
00027 // and secondary (ignoring the third case of "user").
00028 //
00029 // primary resource include readers, writers, catalogs and sometimes
00030 // symbol tables (symtabs).
00031 //
00032 // primary resources are separately allocated and are freed when they
00033 // are closed.
00034 //
00035 // secondary resources are allocated in the scope of a primary resource.
00036 // secondary resources are, for example, ion_strings, decimal and timestamp
00037 // values.  and often symbol tables.
00038 // secodary resources are automattically freed when their associated
00039 // primary resource is freed.
00040 //
00041 // this is handled by making allocating memory in pages, where the
00042 // primary resource is the first page of such a chain.  Then freeing
00043 // the entire chain as a single operation.
00044 //
